## 在本地运行

克隆项目到你的本地

```bash
git clone https://github.com/bryqiu/vue-clean-admin
```

前往项目目录

```bash
cd vue-clean-admin
```

安装依赖

```bash
pnpm i
```

启动服务器

```bash
pnpm dev
```
